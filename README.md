# scoop-bucket

Eigenes [Scoop](https://scoop.sh)-Bucket für Windows-Tools.

## Installation

```powershell
scoop bucket add skquievreux https://github.com/skquievreux/scoop-bucket
scoop install skquievreux/teleprompter
```
